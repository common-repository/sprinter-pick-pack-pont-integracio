Sprinter és Pick Pack Pont csomagküldés integrációja WooCommerce-hez

== Description ==

A bővítmény használatával a Sprinter azon ügyfelei, akik webshopjukat WordPress/WooCommerce platformon üzemeltetik, mind házhozszállítás, mind Pick Pack Pont szolgáltatás esetén a webáruház admin felületéről tudják az adatokat a Sprinternek átküldeni, valamint a megrendeléseikhez a fuvarlevelet (címkét) és a csomagátvételi elismervényt elkészíteni.

Funkciók:

*	Házhoz szállításos csomagok rögzítése
*	Pick Pack Pont csomagok rögzítése
*	Címke és átvételi elismervény kezelés

Követelmények:

*	Jóváhagyott, céges Sprinter - Partner Portál regisztráció
*	Feladáshoz szükséges adatok lekérése a Sprinter IT osztályától a pudointerface@lapker.hu email címen keresztül

ÁSZF:
*	[Sprinter ÁSZF](https://sprinter.hu/aszf/)
*	[Pick Pack Pont ÁSZF](https://pickpackpont.hu/aszf/)

Bővítmény részletesebb leírása:
* 	[Leírás](https://www.sprinter.hu/woocommerce-plugin/)

== Frequently Asked Questions ==
= Honnan kaphatom meg a beállításhoz szükséges információkat? =
A beállításhoz szükséges partnerkód a Partner Portál felületéről érhető el, a prefix és token mezőket a pudointerface@lapker.hu email címen van lehetőség igényelni.

= Mikor állíthatom át a plugin működését 'éles' verzióra a plugin működését? =
Csak a feladáshoz szükséges adatok megadása után valamint a pudointerface@lapker.hu email címről érkező Sprinter IT jóváhagyást követően.
